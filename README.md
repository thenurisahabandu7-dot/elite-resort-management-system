# 🏨 Elite Resort Booking System

A full-stack hotel booking and management system developed using **PHP & MySQL**.

---

## 🌐 Live Demo

👉 https://eliteresort.infinityfreeapp.com

---

## 📌 Features

✔ Dynamic room display on homepage
✔ Admin can add, update, and delete rooms
✔ Image upload for rooms
✔ Booking system with database integration
✔ Admin dashboard with booking analytics
✔ Clean and modern UI design

---

## 🛠 Technologies Used

* PHP
* MySQL
* HTML, CSS, JavaScript
* Chart.js

## ⚙️ Setup Instructions

1. Clone the repository
2. Import the database into phpMyAdmin
3. Update `config.php` with your database details
4. Run on localhost (XAMPP) or deploy online
